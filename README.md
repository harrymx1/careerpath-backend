# CareerPath AI - Backend

## Deskripsi Singkat

Platform asisten karier personal untuk merekomendasikan profesi di bidang IT dan menyusun roadmap belajar adaptif.

## Cara Instalasi & Setup Environment

1. Clone repositori ini.
2. Jalankan `npm install` untuk mengunduh dependensi.
3. Duplikat file `.env.example` menjadi `.env`.
4. Isi kredensial database PostgreSQL di file `.env`.

## Cara Menjalankan Aplikasi

- Jalankan `node src/app.js` untuk memulai server di http://localhost:3000.
